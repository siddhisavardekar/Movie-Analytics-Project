# Movie Analytics Project (ProjectTemplate-based)

## How to Run:

1. Install R packages 
2. Open RStudio
3. Run: library(ProjectTemplate); load.project()
4. Report is in /reports/Analysis_Report.Rmd
5. Plots and models are in /src/03_eda_analysis.R

## Packages Required:
- tidyverse
- lubridate
- ggplot2
- ProjectTemplate
